## Checklist

| Check		| Task
| :--		| :--
|			| Check Hardware (or Virtualbox-settings) 
|			| IP configuration
|			| Services (nmb, smb) are running
|			| Firewall is configured 
|			| Users are created
|			| Samba configuration file is correct
|			| SELinux is configured (booleans: samba_enable_home_dirs, samba_export_all_rw)
